# FortuneAlgorithm

A C++ implementation of the Fortune's algorithm for Voronoi diagram construction.



## Build

The demo requires the [SFML](https://www.sfml-dev.org/) library.

Then you can build the project using [cmake](https://cmake.org/):

```
cmake .
make
```
